CommitCat is a simple timer app that uses adorable pixel-style cats and your latest GitHub commits.
(People often underestimate how much motivation cuteness can provide!)

üìå How to Use

  • Connect your GitHub account (you can also use the app without linking it).
  • Tap the Start button to begin the timer.
  • Code for 30 minutes and push a commit to GitHub.
  • Tap the Stop button, and the cat will let you know whether you succeeded or not!

(Note: The cat moves every time you press a button!)

If you're looking to build a consistent development habit, we hope you'll give CommitCat a try for fun.
Feedback is always welcome!
We'll keep improving it to help developers get a little boost of motivation to get started.
